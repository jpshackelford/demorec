🎬 Presentation Mode

PR #16: Render Marp slides directly in your demos

✨ New Feature

Presentation Mode brings Marp slides into your demorec workflow. Write Markdown, render beautiful slides, and seamlessly mix them with terminal demos—all from a single script.

What This Demonstrates

  • ✓ Vim Editing: Edit a Marp presentation with syntax highlighting
  • ✓ Presentation Mode: Render slides with @mode presentation "file.md"
  • ✓ OpenHands Theme: Professional styling with Set Theme "openhands"
  • ✓ URL Loading: Load presentations from GitHub URLs
  • ✓ Mode Switching: Seamless transitions between slides and terminal

Mode Switches

New Syntax

CommandDescription
@mode presentation "file.md"Switch to presentation mode with local Marp file
@mode presentation "https://..."Load presentation from URL
Set Theme "openhands"Apply theme (alias, URL, or local CSS)
Slide N [time]Display slide N for minimum time

Example Script

# Mix presentations with terminal demos
@mode presentation "intro.md"
Set Theme "openhands"
Slide 1 3s
Slide 2 3s

@mode terminal
Type "pip install mypackage"
Enter

@mode presentation "outro.md"
Slide 1 3s

Use Cases

  • ✓ Product Demos: Intro slides → Live demo → Closing slides
  • ✓ Tutorials: Explain concepts with slides, then demonstrate in terminal
  • ✓ Conference Talks: Record polished presentations with live coding
  • ✓ Documentation: Visual explanations alongside code examples
