# What Is Presentation Mode?

Record beautiful slides alongside your terminal demos

---

# Key Features

- **Marp Integration** - Industry-standard Markdown presentations
- **Theme Support** - Use built-in or custom CSS themes
- **URL Loading** - Load presentations from GitHub URLs
- **Smart Timing** - Slides wait for narration to complete
- **Seamless Mixing** - Switch between slides and terminal
