OpenHands CLI Primitives NEW

High-level commands for recording OpenHands CLI demos

Available Primitives

Command Description
Install Installs OpenHands CLI via uv tool install openhands
Start Launches the CLI and waits for it to be ready
Prompt "text" Sends a single-line prompt to the AI
MultilinePrompt "text" Sends a multi-line prompt (uses Ctrl+L mode)
WaitForReady [timeout] Waits for AI response to complete (default 60s)
Command "/slash" Executes a slash command (e.g., /history)
Quit Exits the CLI cleanly (Ctrl+Q)
Usage: Use @mode terminal:openhands to enable these primitives in your demorec script.
